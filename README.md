<h1 align="center"> Marketing and Sales Analytics Dashboard(Power BI) </h1>

<p align="center">
  A comprehensive Power BI project analyzing <b>Marketing Campaign Performance</b> and <b>Sales Insights</b> using Excel datasets.
</p>

<hr>

<h2> Project Overview</h2>
<p>
This project presents interactive dashboards built using <b>Power BI</b> connected to Excel datasets. 
It focuses on analyzing marketing effectiveness, customer behavior, and sales performance to generate actionable insights.
</p>

<ul>
  <li> Marketing Campaign Analysis</li>
  <li> Sales Performance Tracking</li>
  <li> Customer Segmentation Insights</li>
  <li> Channel Effectiveness Evaluation</li>
</ul>

<h2> Dataset Details</h2>
<p>
The dataset includes key business variables such as:
</p>
<ul>
  <li>Region, Channel, Product Category</li>
  <li>Customer Segment</li>
  <li>Ad Spend & Impressions</li>
  <li>Click-Through Rate (CTR)</li>
  <li>Sales Revenue</li>
  <li>Customer Lifetime Value</li>
</ul>

<hr>

<h2> Key Insights</h2>
<ul>
  <li> Higher ad spend does not always guarantee higher sales</li>
  <li> Social Media and Search channels show strong CTR performance</li>
  <li> Certain product categories generate higher customer lifetime value</li>
  <li> Regional variations significantly impact sales and marketing efficiency</li>
</ul>
